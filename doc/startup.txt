Start memcached:
> memcached

Start the redis server: 
(Make sure redis is installed and redis-server is in your path)
> redis-server

Create a resque worker: 
(You can specify a specific queue, e.g. page_views)
> QUEUE=* rake resque:work

To create a rescue worker with debug messages, run:
> VVERBOSE=1 QUEUE=* rake resque:work

To create multiple rescue workers:
> COUNT=5 QUEUE=* rake resque:workers

To monitor rescue, visit: http://[server_name]/resque
